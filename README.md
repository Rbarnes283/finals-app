# finals-app
It is a simple pregnancy-tracking web application built using HTML, CSS, and JavaScript.

The app allows a user to enter their LMP or conception date, choose units and language, and then view:

Estimated due date

Gestational age

Baby's size each week

Week-by-week development

Recommendations for mom

A printable summary

A shareable link

All calculations and data display happen in the browser. No backend is required.

Files Included

index.html – Main page to enter pregnancy information

development.html – Page that displays development results

styles.css – Styling for the entire app

script.js – Handles calculations, rendering, and interactions

pregnancyData.js – Baby development dataset (weeks 4–40)

pregnant.png – Icon image used in the UI

How to Run

Simply open index.html in any web browser.
The app runs completely on the client side.

Purpose

This project demonstrates:

HTML page structure

CSS layout and design

JavaScript for dynamic content

Passing data between pages

Working with dates and calculations

Clean user interface design
